# Bluestock Mutual Fund Capstone Project

## Overview

This project is part of the Data Analyst Internship Cohort 2025.
The project focuses on Mutual Fund analytics, ETL pipelines, data ingestion, and financial data analysis using Python and SQL.

## Day 1 Tasks Completed

* Project folder structure creation
* Python dependency installation
* CSV dataset ingestion using Pandas
* Live NAV data fetching using mfapi.in API
* ETL pipeline setup
* Data quality validation
* GitHub repository setup

## Technologies Used

* Python
* Pandas
* NumPy
* Requests
* SQL
* Jupyter Notebook
* Git & GitHub
